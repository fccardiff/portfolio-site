About This Project
AutomatedSEO was spun out of an in-house need from Beachgoer.

We were paying quite a bit for traditional PR and media placements, but I figured we could certainly automate portions of the process.

After some serious research, I came up with a basic outline on digital PR -- targeted outreach based on a company's industry. I allowed this to be configurable in the application through the use of seed keywords and APIs.

This data is then filtered through and exported into CSVs for easy readability, and optionally, can be used to generate custom outreach emails to PR prospects.

A base version of the code was uploaded to the GitHub repo, while some more advanced features are kept in-house.
