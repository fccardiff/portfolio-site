CWP Invoicing
Simplifying the invoicing process for a local preschool.
Languages/Libs Used: Node.js, node-csv, json2csv, SendGrid, html-pdf
About This Project
CWP Invoicing came out of a need from a local preschool, who was spending quite a bit of time manually emailing out invoices.

I saw that the process could be simplified to a basic upload-and-send process, using a CSV.

Leveraging this, the script is able to take a CSV as input, as well as line items for invoices, and emails. It'll then batch out the emails with Twilio SendGrid.

This helped reduce employee time by a ton - allowing minor edits to invoice accounts, rather than sending each and every one from the ground up, a process that took several hours.

This code was kept proprietary for the business, but is available upon request.
