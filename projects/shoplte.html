ShopLTE
An AdminLTE fork for niche eCommerce site owners.
Languages/Libs Used: Node.js, HTML, CSS, Shopify API, MongoDB, Javascript, eBay API, Express, Handlebars, ... and dozens more
About This Project
ShopLTE was started as a small pet project - mainly aimed at creating a database to store Shopify orders data, even if we had multiple stores.

Since frontend wasn't a major element, I opted to use Colorlib's AdminLTE and make a simple modified admin dashboard.

Configurability was introduced to allow polling from multiple stores, tagging them as requested and storing into a MongoDB instance.

This data can then be analyzed for profitability trends, analytics, etc. as needed with some basic configuration. View ShopLTE on GitHub
